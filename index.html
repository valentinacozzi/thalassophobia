The Project

The project has been carried out for the Laboratory exam held by Professor Francesca Tomasi for the a.a. 2021-2022, within the Digital Humanities and Digital Knowledge master's degree at the University of Bologna.

The aim of the project is to create a digital object that incorporates 2/3 of the topics that were covered during the course. The topics which have interesed me the most are digital storytelling, digitisation and virtual reconstruction. For the purpose of the present project I took the following lectures as inspirations:

  • Federico Meschini (Università degli Studi della Tuscia), Digital storytelling between narrative and database logic and Electronic publishing and digital storytelling
  • Daniele Ferdani (CNR), Digitisation of Cultural Heritage using image-based modelling techniques
  • Daniele Ferdani (CNR), Virtual restoration and virtual reconstruction in cultural heritage

The Workflow

DIGITAL STORYTELLING WITH TWINE

The digital storytelling was done using Twine, which is an open-source tool for telling interactive, nonlinear stories.

3D RENDER WITH PHOTOGRAMMETRY'S SOFTWARE 3DF ZEPHIR

For the creation of the 3D render of the Reversible Plushie Octopus I used 3DF Zephyr.

I used a full frame Canon 6D and a total of 21 photographs. All the images processed were accepted by the software; I initially obtained a sparse cloud, I cleaned the cloud and discarded all the parts of the images that were irrelevant to the further. I then generated a generated a 3D model, namely the dense cloud model. Ultimately, I generated the final mesh, textured it and exported it in Sketchfab.

Reversible Bilbo by Valentina.Cozzi on Sketchfab

MODELLING AND ANIMATING A STORMY OCEAN WITH BLENDER

For what concerns the stormy ocean animation I created it using Blender and following a YouTube video tutorial.

Firstly, I created the surface starting from a monkey mesh and adding an ocean modifier. I duplicated the ocean modifier and created a larger ocean scene. I set up the camera so as to ensure that it moves with the waves of the ocean; I then created the underwater ocean by adding a plane object to the scene and extruding it downwards. I inserted a particle system for the rain, I added an icosphere and created the raindrops. Thereafter I set up the lighting and the materials: with the Shader editor I added the clouds and the material of the ocean. Finally, I rendered the animation and exported it as a FFmpeg file.
